transition 속성

transiton 속성은 CSS 값이 달라질 때 요소에 어떤 효과로 반영될지 지정합니다.
속성, 지속시간, 시간함수, 지연시간 순으로 값을 넣을 수 있습니다.

주의! 적용할 CSS 속성이 이미 기존값이 적용되어 있어야 동작합니다.
